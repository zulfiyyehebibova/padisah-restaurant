# 👑 Padişah Restoran — Rəqəmsal Menyu və Canlı Səbət Sistemi

---

## 📸 Layihə Haqqında

**Padişah Restoran** — Bu layihə, **NaN Tech** tədris mərkəzindəki intensiv Front-End proqramlaşdırma təhsilinin ilk praktiki imtahan mərhələsi üçün sıfırdan layihələndirilmiş və hazırlanmışdır.

Layihə müvəffəqiyyətlə tamamlanaraq mərkəzin daxili serverində (`nantech.az` subdomeni altında) rəsmi olaraq canlıya buraxılmışdır. Layihənin məqsədi real bazar tələblərinə cavab verən, təmiz kod strukturuna (clean code) malik e-ticarət və qida çatdırılması platformasının funksional alqoritmlərini reallaşdırmaq olmuşdur.

---

## ✨ Xüsusiyyətlər

- 🔍 **Real vaxtda axtarış** — məhsul adına görə anlıq filtrasiya
- 🗂️ **Kateqoriya filtri** — Salat, Pizza, Kabab, Pide, Dessert, İçkilər
- 🛒 **Dinamik səbət** — məhsul əlavə et, sil, ümumi məbləği gör
- 📱 **Responsive dizayn** — mobil, tablet və masaüstü uyğunluğu
- 🍹 **İçki bölməsi** — İsti, Alkolsuz kateqoriyaları ilə
- 🍰 **Dessert bölməsi** — hover effektli kart dizaynı
- 🌙 **Scroll animasiyaları** — Intersection Observer ilə səhifə açılış efektləri
- 🍔 **Mobil menyu** — burger düyməsi ilə açılan nav panel
